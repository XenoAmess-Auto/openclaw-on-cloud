## OOC 平台聊天串房间问题修复

**问题**: 不同群的聊天消息串了（一个房间的消息显示到另一个房间）

**根本原因**: 
当用户切换房间时，WebSocket session 可能由于竞态条件或多次 join 请求，没有正确从旧房间的 session 集合中移除，导致同一个 session 同时存在于多个房间，从而收到多个房间的消息。

**修复方案**:
1. 在 `WebSocketBroadcastService.registerRoomSession()` 中，先调用 `removeSessionFromAllRooms()` 确保 session 从所有其他房间移除，再注册到新房间
2. 添加 `checkForCrossRoomSessions()` 调试日志，用于检测和诊断串房间问题

**修改文件**: `backend/src/main/java/com/ooc/websocket/WebSocketBroadcastService.java`
